Data and Methods

This visualization uses environmental sensor data from the City of Chicago (Source) for January through June 2017. Data was cleaned, filtered, and aggregated using Python (Pandas, GeoPandas) and visualized using Altair (which generates Vega-Lite specifications).

Temperature values for certain sensors (MK-III, Cumulus, TM1) were corrected based on observed patterns and documentation where available (e.g., converting mV readings to Celsius). Outliers and missing values were handled as described in the accompanying report.

Interactive features (selection, brushing) are implemented using Altair's built-in capabilities, linked across different views.
